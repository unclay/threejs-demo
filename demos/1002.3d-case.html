第一个 3D 案例，这是所有的 3D 项目的前置基础

例如：眼睛看着一棵树，在大脑里面构造画面

理解为：相机（眼睛）拍到一个场景（一片空间里面有一棵树），在胶片上里面形成图片（构造画面）

程序是：相机Camera（眼睛） + 场景Scene（空间+一棵树） + 渲染器Render（构造画面） = 屏幕画面

所以本章节是学习三剑客：相机Camera、场景Scene、渲染器Render

demo：我的眼睛看到100米处有一棵树和一个箱子

        // 我的眼睛
        const camera = new THREE.PerspectiveCamera( 75, window.innerWidth / window.innerHeight, 0.1, 1000);
